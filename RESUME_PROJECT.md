# Resume Project Section

## Stock Sentiment Analysis	[Date]

Built a real-time stock market sentiment analysis web application using Flask and vanilla JavaScript, enabling investors to monitor market sentiment through the CNN Fear & Greed Index and analyze sentiment-labeled financial news articles in real-time

Integrated Alpha Vantage and alternative.me APIs to fetch live market data and sentiment scores, implementing intelligent article filtering algorithms that process multiple news sources and classify sentiment labels, reducing manual sentiment analysis time by ~80% through automated ticker sentiment extraction

Designed a responsive user interface with CSS Grid layout supporting dynamic article display across multiple screen sizes, implementing RESTful API endpoints with CORS support and comprehensive error handling to achieve <2 second API response times and ensure reliable real-time data visualization
